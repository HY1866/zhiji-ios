# 智记 - 时间点记录器

一款 Linux 桌面悬浮工具，用于快速记录时间点，支持 AI 语音识别备注。基于 Python 3 + GTK3 构建，采用讯飞在线识别 + faster-whisper 离线识别双引擎实现高精度中文语音识别。

## 功能特性

- **一键记录当前时间点** — 点击按钮即刻记录，无需手动输入
- **语音识别记录** — 讯飞在线识别 + faster-whisper 离线识别双引擎，支持中文，自动将语音转为文字备注
- **多口音支持** — 支持普通话、四川话、河南话三种口音，根据离线库学习情况动态调整识别优先级
- **AI 自动学习离线库** — 词或短语出现 3 次及以上自动入库，按口音分类存储，支持相似语音自动匹配
- **修改记录自动学习** — 修改识别结果后自动学习，后续识别自动应用修正
- **智能静音检测** — 最长录制15秒，静音5秒自动停止，无需手动操作
- **时间格式切换** — 支持中文格式（2026年5月14日）和ISO格式（2026-05-14）
- **识别模式切换** — 在线/离线识别一键切换，自动测试新模式可用性
- **文本后处理** — 英文字母自动转大写，关键词前自动添加逗号
- **帮助按钮** — 内置使用说明对话框，支持翻页浏览
- **深色主题现代界面** — 精心设计的暗色 UI，长时间使用不疲劳
- **悬浮窗口，始终置顶** — 不遮挡工作，随时可用
- **可调整窗口大小** — 拖拽窗口边缘调整大小，自动保存位置和尺寸
- **全屏模式** — 一键切换全屏，文本可编辑，便于查看和修改记录
- **系统托盘图标** — 点击程序图标即可启动/显示程序，右键菜单含重置窗口大小功能
- **记录查看与 CSV 导出** — 支持历史记录浏览，左键选中，右键编辑，一键导出 CSV 文件
- **修改记录支持语音输入** — 编辑时可点击语音输入，自动保留原时间戳
- **防止重复运行** — 确保同一时间只有一个实例运行
- **日志记录功能** — 所有识别过程保存到 Log 文件夹，自动清理2天前的旧日志
- **多语音队列识别** — 上条识别未完成时可继续录制，自动排队依次识别
- **快捷键支持** — Ctrl+S保存（全屏时），Esc退出全屏
- **高对比度按钮** — 删除/取消/确定按钮样式醒目，易于区分
- **分层顶部布局** — 帮助/全屏按钮在第一排，时间格式/时钟/计数在第二排

## 使用方法

### 启动程序

- **常规启动**：双击桌面图标运行
- **手动安装**：在终端执行 `python3 zhiji.py`

### 基本操作

- **📌 记录** — 点击记录按钮，立即保存当前时间点到文件
- **🎤 语音** — 点击语音按钮开始录音，最长15秒，静音5秒自动停止；AI 自动识别语音内容作为备注；支持连续录音排队识别
- **� 模式切换** — 点击模式按钮切换在线/离线识别，自动测试新模式可用性
- **� 格式按钮** — 切换时间显示格式（中文/ISO）
- **❓ 帮助按钮** — 查看使用说明（支持翻页浏览）
- **↗️ 全屏按钮** — 一键切换全屏模式，文本可编辑
- **📋 查看** — 查看已记录的所有时间点
- **📤 导出** — 将记录导出为 CSV 格式文件
- **📚 查看离线库** — 右键点击日志区域，选择「查看离线库」查看已学习的词/句（支持翻页浏览）
- **✏️ 编辑记录** — 左键选中记录，右键点击选择「修改此条记录」；编辑时可语音输入，自动保留原时间戳
- **📐 调整窗口** — 拖拽窗口边缘调整大小，自动保存
- **⌨️ 快捷键** — Ctrl+S保存（全屏时），Esc退出全屏

### 系统托盘操作

- **左键点击** — 显示/隐藏主窗口
- **右键点击** — 弹出菜单，可选择「显示窗口」、「时间格式」、「重置窗口大小」、「使用说明」或「退出」程序
- **窗口关闭** — 点击窗口关闭按钮时，程序最小化到托盘，而非完全退出
- **任务栏** — 窗口会显示在任务栏中，可切换

### 数据存储

所有记录保存在项目目录 `Document Dashboard/时间记录.txt` 文件中，纯文本格式，方便查看和备份。

配置和离线库保存在项目目录的 `Offline data/` 子目录下：
- `zhiji_config.json` - 程序配置（时间格式、窗口大小、窗口位置等）
- `智记_离线库.json` - AI离线学习库，词/短语出现3次及以上自动学习，修改记录自动学习修正，支持相似语音自动匹配，按口音分类存储。

日志保存在项目目录的 `Log/` 子目录下：
- 每次识别生成一个带时间戳的日志文件
- 自动保留最近2天的日志，旧日志自动清理

辅助文件保存在项目目录的 `Document Dashboard/` 子目录下：
- `Linux兼容性分析报告.md` - Linux兼容性详细分析
- `时间记录.txt` - 时间记录文件

## 语音识别说明

- **双引擎识别**：
  - **讯飞在线识别**（主引擎）：使用讯飞 WebSocket API，快速准确
  - **faster-whisper 离线识别**（备选引擎）：当讯飞不可用时自动切换
- **模式切换**：可手动切换在线/离线识别模式，切换时自动测试新模式可用性
- **多口音支持**：支持普通话、四川话、河南话三种口音
- **动态优先级**：根据离线库各口音学习词汇数量，动态调整识别尝试顺序，优先尝试最常用的口音
- **队列识别**：支持连续录音，上条识别未完成时新录音自动排队
- 使用 **faster-whisper** 引擎搭配 **small** 模型
- 首次使用离线识别时会自动下载模型文件（约 **461MB**），下载后缓存在 `Offline data/whisper-models/` 文件夹，后续启动无需重复下载
- 对中文语音识别有较高准确率
- 内置智能静音检测，最长15秒，静音5秒自动停止并触发识别

## 常见问题

### 麦克风无法录音

1. 确认系统使用 PulseAudio 音频服务：
   ```bash
   pactl info
   ```
2. 检查麦克风是否被识别：
   ```bash
   pactl list sources short
   ```
3. 确认麦克风未被静音：
   ```bash
   pactl set-source-mute @DEFAULT_SOURCE@ 0
   ```

### 模型下载失败或速度慢

- faster-whisper 模型从 Hugging Face 下载，若网络受限，可尝试设置代理：
  ```bash
  export https_proxy=http://your-proxy:port
  python3 -c "from faster_whisper import WhisperModel; WhisperModel('small', device='cpu', compute_type='int8', download_root='Offline data/whisper-models')"
  ```
- 模型缓存目录在 `Offline data/whisper-models/`，可检查该目录确认下载状态。

### 提示 ffmpeg 未找到

确保 ffmpeg 已安装并在 PATH 中：
```bash
ffmpeg -version
```
若未安装，使用包管理器安装：
```bash
# Ubuntu/Debian
sudo apt install -y ffmpeg

# Fedora
sudo dnf install -y ffmpeg

# Arch
sudo pacman -S --noconfirm ffmpeg
```

### GTK 相关错误

若运行时出现 `Gtk-WARNING` 或 `gi.repository` 导入错误，确认已安装 GTK3 相关包：
```bash
# Ubuntu/Debian
sudo apt install -y python3-gi python3-gi-cairo gir1.2-gtk-3.0

# Fedora
sudo dnf install -y python3-gobject gtk3

# Arch
sudo pacman -S --noconfirm python-gobject gtk3
```

### 如何查看识别日志

日志文件保存在 `Log/` 文件夹下，文件名格式为 `recognize_YYYYMMDD_HHMMSS.log`，每次识别生成一个新文件。

### 如何调整窗口大小

- **拖拽调整**：直接拖拽窗口边缘即可调整大小，调整后自动保存
- **重置大小**：右键点击托盘图标，选择「重置窗口大小」恢复默认尺寸

### 如何使用全屏模式

- **切换全屏**：点击窗口右上角的全屏按钮（↗️）
- **退出全屏**：按 Esc 键或再次点击全屏按钮
- **编辑文本**：全屏模式下文本可直接编辑，按 Ctrl+S 保存
- **语音输入**：全屏模式下点击时间戳可触发语音输入

### 如何使用队列识别

- **连续录音**：上一条语音正在识别时，可继续点击「语音」按钮录音
- **自动排队**：新录音会自动加入队列，等待上一条识别完成后依次处理
- **状态显示**：状态栏会显示队列中剩余任务数量

### 如何切换识别模式

- **切换模式**：点击「在线」或「离线」按钮切换模式
- **自动测试**：切换时会自动测试新模式可用性，如有异常会提示并退回原模式
- **优先使用**：根据需要选择在线（快速）或离线（无需网络）模式

### 编辑记录时如何使用语音输入

- **点击语音按钮**：在编辑对话框中点击「🎤」按钮
- **开始录音**：自动开始录音，最长15秒，静音5秒自动停止
- **保留时间戳**：语音输入只会替换备注内容，原时间戳不会改动
- **自动学习**：保存修改后会自动学习到离线库
