## 📋 完整步骤（首次使用）

### 第1步：配置Git用户（只需一次）

**双击：** `git-config.bat`

### 第2步：设置远程仓库（只需一次）

**双击：** `git-setup.bat`

输入：`https://github.com/HY1866/zhiji-ios.git`

### 第3步：推送到远程

**双击：** `git-initial-push.bat` （首次）

或 **双击：** `一键推送-最简单.bat` （之后）

### 第4步：验证

**双击：** `快速验证.bat`

或访问：https://github.com/HY1866/zhiji-ios

## 🍎 iOS设备部署

### 在Mac上的操作：
```bash
# 1. 克隆项目
cd ~/Desktop
git clone https://github.com/HY1866/zhiji-ios.git
cd zhiji-ios

# 2. 运行设置脚本
./mac-setup.sh

# 3. 打开Xcode
cd ios
open ZhiJi.xcodeproj
```

### 在Xcode中：
1. 选择Team（您的Apple账号）
2. 连接iPhone/iPad
3. 选择您的设备
4. 点击播放按钮（▶️）

**详细指南：** 
- 查看 `iOS设备部署快速参考.md`（快速参考）
- 查看 `iOS部署到设备指南.md`（完整指南）
